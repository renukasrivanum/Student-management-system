# Student Management System

A Python project for managing student records using Object-Oriented Programming (OOP) and JSON file storage.

## Features

* Add Student
* Search Student
* View Students
* Delete Student
* Save Data to JSON File
* Load Data from JSON File

## Concepts Used

* Python Classes and Objects
* Dictionaries
* Loops
* Exception Handling
* File Handling
* JSON

## How to Run

1. Install Python.
2. Open terminal in project folder.
3. Run:

python student_management.py
